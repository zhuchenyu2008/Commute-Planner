# 首页页面规格

## 页面目标

首页是一句话规划入口，也是用户打开应用后的默认工作台。它负责让用户用自然语言创建通勤规划，查看当前活跃行程，并快速进入详情、历史和设置。

## 用户任务

- 输入一句话创建行程，例如“明天 9:15 到龙湖天街”。
- 查看当前城市、天气和系统可用状态。
- 查看当前活跃行程的下一步、剩余时间和提醒状态。
- 使用常用目的地快速发起规划。
- 处理 Agent 发现的待确认记忆提示。

## 全局约定

- 未登录访问首页时跳转到登录门禁，登录成功后返回首页。
- 认证过期时清空当前页面请求状态并回到登录页，保留本地输入草稿。
- 底部导航展示：首页、历史、设置；首页 tab 高亮。
- 视觉沿用 `Lumina Velocity`：浅色地图背景、玻璃态面板、Inter 字体、Commute Blue 主操作色。
- 页面以移动端为第一优先级，桌面端使用居中最大宽度工作台，不改成营销页。

## 布局结构

1. 顶部状态区
   - 左侧显示当前城市和定位来源。
   - 右侧显示天气摘要、温度、天气图标。
   - API key 缺失或外部服务不可用时，用紧凑状态提示替代天气卡。

2. 一句话规划区
   - 主输入框固定为页面视觉焦点。
   - 占位文案：“你想去哪儿？例如 明天 9:15 到龙湖天街”。
   - 左侧使用搜索图标，右侧保留麦克风按钮占位。
   - 提交按钮可集成在键盘回车、输入框右侧或下方主按钮中。

3. Agent 进度区
   - 仅在规划中显示。
   - 分阶段展示：理解需求、定位目的地、计算路线、设置提醒。
   - 每个阶段使用短文本和状态点，避免长解释。

4. 当前行程区
   - 有活跃行程时展示目的地、路线类型、剩余时间、下一步和“查看详情”按钮。
   - 无活跃行程时隐藏该区或显示轻量空状态，不制造大面积空卡片。

5. 待确认记忆提示
   - 当 Agent 发现新偏好但尚未确认时显示。
   - 文案示例：“发现 2 条待确认偏好”。
   - 操作：查看、稍后；查看跳转设置页的待确认记忆分组。

6. 常用目的地区
   - 展示已确认的 `place` 或 `alias` 记忆，例如家、公司、学校、健身房。
   - 每项显示名称、预计耗时或最近一次路线状态。

7. 底部导航
   - 首页、历史、设置。
   - 详情页不作为底部主 tab，从当前行程或历史记录进入。

## 页面状态

- `unauthenticated`：跳转登录页。
- `loading`：加载 profile、天气和活跃行程。
- `readyEmpty`：无活跃行程。
- `readyWithActiveTrip`：存在活跃行程。
- `planning`：Agent 正在处理输入。
- `ambiguousDestination`：目的地候选不明确，需要用户选择或补充。
- `pastArrivalTime`：用户要求的到达时间已过期，不创建提醒。
- `serviceMissingConfig`：高德、模型、Telegram 或邮件配置缺失。
- `offline`：网络不可用或后端不可达。
- `authExpired`：认证过期，跳转登录页。

## 核心交互

- 用户提交自然语言后调用 `POST /api/agent/messages`。
- 规划过程中禁用重复提交，但允许用户取消当前输入。
- 规划成功后跳转到 `/trips/{id}` 详情页。
- 如果 Agent 返回待确认记忆，首页只展示轻量提示，不在首页展开完整记忆管理。
- 点击当前行程卡片或“查看详情”进入详情页。
- 点击常用目的地后，将目的地写入输入框并触发一次规划前确认态；用户可直接提交。
- 麦克风按钮初版为占位入口，点击后提示“语音输入稍后支持”，不阻断主流程。

## 接口

### `GET /api/profile`

用于加载默认城市、默认起点、时区、用户显示偏好和基础配置状态。

### `GET /api/trips/active`

用于加载当前活跃行程。没有活跃行程时返回空值，而不是错误。

### `POST /api/agent/messages`

请求示例：

```json
{
  "text": "明天 9:15 到龙湖天街",
  "source": "web"
}
```

成功响应需要包含：

```json
{
  "message": "已为你规划好路线，最晚 08:10 出发。",
  "tripId": "trip_123",
  "pendingMemoryCount": 1
}
```

## 验收点

- 未登录访问首页会进入登录门禁，登录后返回首页。
- 首页可以在无活跃行程、有活跃行程、规划中、错误四类状态下稳定展示。
- 输入“明天 9:15 到龙湖天街”会调用 Agent 接口，成功后进入详情页。
- 待确认记忆提示不会抢占主输入体验，可以跳转设置页处理。
- 首页不展示密钥明文，不暴露内部错误堆栈。
